MEMENTO

La ligne de commande avec le terminal: Linux ubuntu

Quelque commande

  • Afficher le manuel pour la commande ls: man ls
  • Afficher le contenu dossier courant utiliser la commande: ls
  • Afficher le contenu dossier courant et fichiers cachés: ls -a
  • Afficher le contenu dossier version longue sans les fichiers cachés: ls -l
  • Afficher le contenu dossier version longue avec les fichiers cachés: ls -al
  • Afficher le chemin dossier courant: pwd
  • Se déplacer dans le dossier Documents: cd Documents
  • Afficher le contenu de ce dossier: ls

Super utilisateur et gestion de paquets

  • Commande pour se mettre en super utilisateur: sudo -s
    • Puis mettre son mot de passe utilisateur

    Gérer des paquets c'est à dire mettre à jours certaine fonctionnalité et logiciel.

  • Mettre à jour la liste des paquets: apt update
  • Mettre à jour la liste des paquets: apt update
  • Mettre à jour les paquets: apt upgrade
  • Trouver les paquets contenant le mot( ex: XML ): apt search XML
  • Pour installer le paquet git: install git
  • Télécharger un logiciel par internet: ~# cd Téléchargements/
    • ~/Téléchargements # dpkg -i (logiciel a installer).deb (.deb avec debian)
  • Pour supprimé un logiciel: apt remove en mode super utilisateur.

Création de dossiers et fichiers

  • Créer un dossier website dans documents: cd documents
    • /cd documents $ mkdir website
  • Créer un dossier et un sous dossier en une seule ligne de commande dans le dossier website: Mémento/assets
    • mkdir -p website/Memento/assets
  • Se déplacer dans le dossier nouvellement créer et faire 3 nouveaux dossiers en une seule ligne de commande: css, img, js.
    • /Documents $ cd website/Memento/assets /Documents/website/Memento/assets $ mkdir css img js
  • Créer un fichier style.css dans Memento: /Documents/website/Memento $ touch style.css

    Créer un fichier index.html

  • Déplacer un fichier: mv style.css assets/css
  • mv (nom du fichier extension destination)
  • mv permet de déplacer un fichier, il etait dans Memento et on le déplace vers assets/css
  • mv permet aussi de renomé des fichiers ou dossiers: exemple: mv style.css design.css
